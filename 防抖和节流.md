# 防抖和节流的总结和实践

## 1. 防抖函数

- 实现原理

触发高频事件后n秒内函数只会执行一次，如果n秒内高频事件再次被触发，则重新计算时间

- 使用场景

高频事件触发，但在n秒内只会执行一次，所以节流会稀释函数的执行频率,如：
1、按钮提交场景
2、服务端验证场景

- 代码

```
function debounce(func, wait) {
    var timeout;
    return function () {
        var context = this;
        var args = arguments;
        clearTimeout(timeout)
        timeout = setTimeout(function(){
            func.apply(context, args)
        }, wait);
    }
}
```

## 2. 节流函数

- 实现原理

每次触发事件时都判断当前是否有等待执行的延时函数

- 使用场景
1、拖拽场景
2、缩放场景
3、动画场景
- 代码

```
function throttle(func, wait) {
    var timeout;
    var previous = 0;

    return function() {
        context = this;
        args = arguments;
        if (!timeout) {
            timeout = setTimeout(function(){
                timeout = null;
                func.apply(context, args)
            }, wait)
        }

    }
}
```
